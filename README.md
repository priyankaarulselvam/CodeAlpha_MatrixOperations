# CodeAlpha - Matrix Operations

A simple C programming project to perform basic operations on matrices, created for the CodeAlpha internship.

## Features
* *Matrix Addition* - Adds two matrices of the same size.
* *Matrix Multiplication* - Multiplies two matrices after checking dimensions.
* *Matrix Transpose* - Swaps rows and columns of a matrix.

---

## Real-Life Example
Think of matrix addition like tracking sales for *two branches of a shop*. If Row 1 represents days and Column 1 represents items sold, adding Matrix A (Shop 1) and Matrix B (Shop 2) gives you the total sales for your whole business.
